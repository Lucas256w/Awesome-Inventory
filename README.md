## Features
- Create, Read, Update, and Delete Categories and Items
- View all Items in a Category
- Persistent data storage in a database

## Made with

-   Express with NodeJS
-   MongoDB with Mongoose ODM
-   MVC design pattern
-   Hosted on Railway
